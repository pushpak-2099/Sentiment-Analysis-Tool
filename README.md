# Sentiment Analysis Tool

This is a simple Python-based sentiment analysis tool that analyzes
user input and classifies it as Positive, Negative, or Neutral.

## Features
- Takes text input from the user
- Analyzes sentiment using TextBlob
- Displays sentiment result
- Runs in a loop until "stop" is entered

## Technologies Used
- Python
- TextBlob (NLP Library)

## How to Run

1. Install TextBlob:
   pip install textblob

2. Run the program:
   python sentiment_analysis.py

3. Enter your text and get the sentiment result.

## Example

Input:
I love learning AI and ML

Output:
The sentence is positive.
